## 内容如何跟 Dialog 本身交互

### 使用 hook `useDialog`

```typescript
import {
  useDialog
} from '@alicloud/console-base-rc-dialog';

function SomeComponent() {
  const {
    close,
    lock,
    unlock
  } = useDialog();
}
```

### 使用 Context

```typescript
import {
  DialogContext
} from '@alicloud/console-base-rc-dialog';
```
